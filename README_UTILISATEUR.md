# Guide d'installation et d'utilisation de l'application

## Installation

1. **Téléchargement**
   - Récupérez les fichiers `consultation_article.exe` et `article.db` fourni ou téléchargez-le à partir du lien communiqué.

2. **Emplacement**
   - Placez les fichiers `consultation_article.exe` et `article.db` dans le dossier de votre choix sur votre ordinateur (par exemple sur le Bureau ou dans `Mes Documents`).

3. **Premier lancement**
   - Double-cliquez sur le fichier `consultation_article.exe` pour démarrer l’application.
   - Si une alerte de sécurité Windows apparaît, cliquez sur « Informations complémentaires » puis sur « Exécuter quand même ».

## Utilisation

1. **Ouverture de l’application**
   - Après double-clic, l’interface de l’application s’ouvre automatiquement.

2. **Fonctionnalités principales**
   - **Consulter des articles** : Parcourez la liste des articles disponibles et cliquez sur un article pour afficher son contenu.
   - **Recherche** : Utilisez la barre de recherche pour trouver rapidement un article par mot-clé.

3. **Quitter l’application**
   - Cliquez sur le bouton en haut à droite pour fermer l’application.

## Conseils et dépannage

- Si l’application ne démarre pas, vérifiez que votre antivirus ne bloque pas le fichier `.exe`.
- Si vous voyez un message d’erreur, contactez francis.khaled-khodja@tdf.fr ou la personne qui vous a fourni l’application.
- Le fichier de base de données `   article.db` est nécessaire : il doit être placé dans le même dossier que le fichier `consultation_article.exe`.
